# Dossier Vidéo

Placez votre vidéo de présentation des templates dans ce dossier.

## Nom du fichier
- Nom recommandé : `presentation-templates.mp4`
- Formats supportés : MP4, WebM

## Formats recommandés
- **Format vidéo** : MP4 (H.264)
- **Résolution** : 1920x1080 (Full HD) ou 1280x720 (HD)
- **Taille** : Essayez de garder la vidéo sous 50-100 MB pour un chargement rapide

## Alternative : Vidéo YouTube
Si vous préférez utiliser une vidéo YouTube, vous pouvez modifier le code dans `src/views/Templates.vue` pour utiliser un iframe YouTube à la place.
