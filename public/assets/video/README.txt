==========================================
DOSSIER POUR LA VIDÉO DE PRÉSENTATION
==========================================

Placez votre vidéo de présentation des templates dans ce dossier.

NOM DU FICHIER RECOMMANDÉ :
- presentation-templates.mp4

FORMATS SUPPORTÉS :
- MP4 (recommandé)
- WebM (optionnel)

SPÉCIFICATIONS RECOMMANDÉES :
- Format : MP4 (H.264)
- Résolution : 1920x1080 (Full HD) ou 1280x720 (HD)
- Taille : Essayez de garder sous 50-100 MB pour un chargement rapide

ALTERNATIVE : VIDÉO YOUTUBE
Si vous préférez utiliser une vidéo YouTube, vous pouvez modifier 
le code dans src/views/Templates.vue pour utiliser un iframe YouTube.
